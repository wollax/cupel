<objective>
Implement QuotaSlice — a decorator slicer that enforces percentage-based semantic quotas per ContextKind. Includes QuotaSet (immutable config) and QuotaBuilder (fluent builder with validation).

Purpose: QuotaSlice enforces semantic balance in context selection. Without quotas, a high-scoring Kind can dominate the budget. QuotaSlice ensures minimum representation (Require) and prevents over-representation (Cap) per Kind.

Output: Three types (QuotaSet, QuotaBuilder, QuotaSlice) in `Wollax.Cupel.Slicing` namespace with full TDD test coverage. SLICE-04 and SLICE-06 requirements.
</objective>

  <implementation>
    QuotaSlice (namespace Wollax.Cupel.Slicing):
    - Sealed class implementing ISlicer
    - Constructor: `QuotaSlice(ISlicer innerSlicer, QuotaSet quotas)` — validates non-null

    Algorithm:
    1. Early return: if scoredItems.Count == 0 or budget.TargetTokens <= 0, return []

    2. Partition candidates by ContextKind:
       ```
       var partitions = new Dictionary<ContextKind, List<ScoredItem>>();
       for (var i = 0; i < scoredItems.Count; i++)
       {
           var kind = scoredItems[i].Item.Kind;
           if (!partitions.TryGetValue(kind, out var list))
           {
               list = new List<ScoredItem>();
               partitions[kind] = list;
           }
           list.Add(scoredItems[i]);
       }
       ```

    3. Calculate per-Kind token budgets:
       a. For each Kind in quotas.Kinds: compute requireTokens = (int)(quotas.GetRequire(kind) / 100.0 * budget.TargetTokens)
       b. For each Kind in quotas.Kinds: compute capTokens = (int)(quotas.GetCap(kind) / 100.0 * budget.TargetTokens)
       c. Compute totalRequired = sum of all requireTokens
       d. unassignedBudget = budget.TargetTokens - totalRequired
       e. Distribute unassigned proportional to each Kind's candidate token mass (sum of tokens in that Kind's partition). Kinds at their cap get no more. Unconfigured kinds share the unassigned pool.

    4. Per-Kind slicing: for each Kind partition, create a sub-budget with:
       - targetTokens = kindBudget (require + proportional unassigned, capped at capTokens)
       - maxTokens = capTokens (or kindBudget if no cap)
       Call innerSlicer.Slice(kindPartition, kindSubBudget, traceCollector)

    5. Merge all per-Kind results into a single list. Return.

    6. If a Kind has Require > 0 but insufficient items, emit trace warning via traceCollector.RecordItemEvent.
