<action>
**ContextKind** (newtype-as-string pattern):

1. Add conditional import at the top of `context_kind.rs`:
   ```rust
   #[cfg(feature = "serde")]
   use serde::{Deserialize, Deserializer, Serialize, Serializer};
   ```

2. Add a manual `Serialize` impl gated behind `#[cfg(feature = "serde")]`:
   ```rust
   #[cfg(feature = "serde")]
   impl Serialize for ContextKind {
       fn serialize<S: Serializer>(&self, serializer: S) -> Result<S::Ok, S::Error> {
           serializer.serialize_str(&self.0)
       }
   }
   ```

3. Add a manual `Deserialize` impl that routes through `ContextKind::new()`:
   ```rust
   #[cfg(feature = "serde")]
   impl<'de> Deserialize<'de> for ContextKind {
       fn deserialize<D: Deserializer<'de>>(deserializer: D) -> Result<Self, D::Error> {
           let s = String::deserialize(deserializer)?;
           ContextKind::new(s).map_err(serde::de::Error::custom)
       }
   }
   ```

**ContextSource** (identical newtype-as-string pattern):

1. Add the same conditional import block.
2. Add manual `Serialize` impl serializing as `&self.0`.
3. Add manual `Deserialize` impl routing through `ContextSource::new()` with `map_err(serde::de::Error::custom)`.

**OverflowStrategy** (simple cfg_attr derive):

1. Change the derive line to:
   ```rust
   #[derive(Debug, Clone, Copy, Default, PartialEq, Eq, Hash)]
   #[cfg_attr(feature = "serde", derive(serde::Serialize, serde::Deserialize))]
   #[cfg_attr(feature = "serde", serde(deny_unknown_fields))]
   pub enum OverflowStrategy {
   ```

Note: PascalCase is the serde default for enum variants, so no `rename_all` needed.
</action>

</done>
