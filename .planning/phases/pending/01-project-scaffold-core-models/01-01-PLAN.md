    2. Create `Directory.Build.props` at repo root with:
       - TargetFramework: net10.0
       - LangVersion: latest
       - Nullable: enable
       - ImplicitUsings: enable
       - TreatWarningsAsErrors: true
       - EnforceCodeStyleInBuild: true
       - NuGet metadata (Authors: Wollax, MIT license, GitHub URLs, PackageTags)
       - Deterministic build settings (ContinuousIntegrationBuild conditional on CI, EmbedUntrackedSources, IncludeSymbols, snupkg)

    3. Create `Directory.Packages.props` with ManagePackageVersionsCentrally=true:
       - GlobalPackageReference: MinVer 7.0.0 (PrivateAssets=All)
       - PackageVersion: Microsoft.SourceLink.GitHub 10.0.102
       - PackageVersion: Microsoft.CodeAnalysis.PublicApiAnalyzers 3.3.4
       - PackageVersion: TUnit 1.19.22
       - PackageVersion: BenchmarkDotNet 0.15.8

    4. Create `src/Wollax.Cupel/Wollax.Cupel.csproj`:
       - SDK-style classlib (no OutputType needed — classlib is default)
       - Description: "Context management library for coding agents. Select optimal context windows with scoring, slicing, and placement."
       - PackageReference: Microsoft.CodeAnalysis.PublicApiAnalyzers (PrivateAssets=All)
       - AdditionalFiles: PublicAPI.Shipped.txt, PublicAPI.Unshipped.txt
       - IsPackable: true (default, but explicit)

    5. Create both `PublicAPI.Shipped.txt` and `PublicAPI.Unshipped.txt` with `#nullable enable` as the first and only line.

    6. Create a minimal `.editorconfig` at repo root with standard C# settings (indent_size 4, end_of_line lf, insert_final_newline true).

    7. Create a placeholder class in the core project so it compiles (e.g., an internal static class or just delete Class1.cs if generated).
