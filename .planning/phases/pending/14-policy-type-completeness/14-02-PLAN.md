With reflection-based derivation:

```csharp
private static readonly string[] BuiltInScorerTypes = Enum.GetValues<ScorerType>()
    .Select(t => typeof(ScorerType)
        .GetField(t.ToString())!
        .GetCustomAttribute<System.Text.Json.Serialization.JsonStringEnumMemberNameAttribute>()!
        .Name)
    .ToArray();
```
