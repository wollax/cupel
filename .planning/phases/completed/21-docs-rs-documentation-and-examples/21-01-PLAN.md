  <action>
  **lib.rs changes:**
  Add these lines at the TOP of lib.rs, before the module declarations:
  ```rust
  #![doc = include_str!("../README.md")]
  #![cfg_attr(docsrs, feature(doc_auto_cfg))]
  ```
  The `doc = include_str!` pulls the README as crate-level documentation.
  The `cfg_attr(docsrs, ...)` enables automatic feature-gate badges on docs.rs.

  Do NOT modify any existing module declarations or re-exports.

  **Cargo.toml changes:**

     ```toml
     [package.metadata.docs.rs]
     all-features = true
     rustdoc-args = ["--cfg", "docsrs"]
     ```
