<objective>
Create the `Wollax.Cupel.Tiktoken` companion package bridging `Microsoft.ML.Tokenizers` for accurate token counting on `ContextItem` content.

Purpose: Cupel's core library requires callers to provide pre-counted token values. This companion package eliminates guesswork by providing accurate tiktoken-based counting via the official Microsoft tokenizer.

Output: A new packable project with `TiktokenTokenCounter` adapter, a test project validating counting accuracy, and updated solution/package config.
</objective>
