  <action>
    1. Create `src/scorer/mod.rs`:
       - Define pub trait Scorer: `fn score(&self, item: &ContextItem, all_items: &[ContextItem]) -> f64;`
       - Re-export all scorer types

    2. Create `src/scorer/recency.rs` — RecencyScorer:
       - Unit struct, implements Scorer
       - Algorithm per spec `RECENCY-SCORE`: count items with timestamp, compute rank (count of items with strictly earlier timestamp), return rank / (countWithTimestamp - 1)
       - No timestamp → 0.0. Single timestamped item → 1.0.

    3. Create `src/scorer/priority.rs` — PriorityScorer:
       - Same rank-based algorithm as Recency but on priority field
       - No priority → 0.0. Single prioritized item → 1.0.

    4. Create `src/scorer/kind.rs` — KindScorer:
       - Constructed with HashMap<ContextKind, f64> weight map
       - `with_default_weights()` constructor: SystemPrompt=1.0, Memory=0.8, ToolOutput=0.6, Document=0.4, Message=0.2
       - `new(weights)` validates non-negative, finite weights
       - Score = weight lookup by item.kind, default 0.0

    5. Create `src/scorer/tag.rs` — TagScorer:
       - Constructed with HashMap<String, f64> tag weights (case-SENSITIVE keys per spec)
       - Precompute total_weight at construction
       - Score = min(matched_sum / total_weight, 1.0). No tags or total_weight=0 → 0.0
       - Validate non-negative, finite weights at construction

    6. Create `src/scorer/frequency.rs` — FrequencyScorer:
       - Unit struct, implements Scorer
       - Score = count of peers sharing any tag / (len - 1)
       - Self-exclusion via `std::ptr::eq(item, other)` — reference identity
       - Tag comparison: CASE-INSENSITIVE ASCII (`eq_ignore_ascii_case`)
       - No tags or single item → 0.0

    7. Create `src/scorer/reflexive.rs` — ReflexiveScorer:
       - Unit struct, implements Scorer
       - Score = clamp(futureRelevanceHint, 0.0, 1.0). null → 0.0. Non-finite → 0.0 (check finiteness BEFORE clamp).

    8. Create `src/scorer/composite.rs` — CompositeScorer:
       - Constructed with Vec<(Box<dyn Scorer>, f64)> entries
       - Validates: at least one entry, all weights positive and finite
       - Normalizes weights at construction (weight_i / sum)
       - Cycle detection via DFS with reference identity (use raw pointer casts for identity comparison of Box<dyn Scorer>)
       - Score = sum(child.score() * normalized_weight)

    9. Create `src/scorer/scaled.rs` — ScaledScorer:
       - Wraps a Box<dyn Scorer> inner
       - Score: iterate all_items, score each with inner, find min/max, identify self via std::ptr::eq
       - If max == min → 0.5. Empty allItems → 0.5.
       - Return (rawScore - min) / (max - min)

    10. Update `src/lib.rs` to include scorer module and re-exports.
