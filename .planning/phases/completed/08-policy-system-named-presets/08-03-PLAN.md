  <action>
**Add WithPolicy() to PipelineBuilder and test it.**

1. Add `WithPolicy(CupelPolicy policy)` method to `PipelineBuilder`:
   - Validate `policy` is not null (`ArgumentNullException.ThrowIfNull`)
   - Iterate `policy.Scorers` and call `AddScorer(CreateScorer(entry), entry.Weight)` for each
   - Resolve slicer: `policy.SlicerType` switch to `UseGreedySlice()` / `UseKnapsackSlice(policy.KnapsackBucketSize ?? 100)`
   - Resolve placer: `policy.PlacerType` switch to `WithPlacer(new ChronologicalPlacer())` / `WithPlacer(new UShapedPlacer())`
   - Apply `WithDeduplication(policy.DeduplicationEnabled)`
   - Apply `WithOverflowStrategy(policy.OverflowStrategy)`
   - Resolve quotas if `policy.Quotas is { Count: > 0 }`:
     ```csharp
     WithQuotas(qb =>
     {
         foreach (var q in policy.Quotas)
         {
             if (q.MinPercent.HasValue) qb.Require(q.Kind, q.MinPercent.Value);
             if (q.MaxPercent.HasValue) qb.Cap(q.Kind, q.MaxPercent.Value);
         }
     });
     ```
   - Return `this` for chaining

2. Add `private static IScorer CreateScorer(ScorerEntry entry)` method:
   - Switch on `entry.Type`:
     - `Recency` -> `new RecencyScorer()`
     - `Priority` -> `new PriorityScorer()`
     - `Kind` -> `entry.KindWeights is not null ? new KindScorer(entry.KindWeights) : new KindScorer()`
     - `Tag` -> `new TagScorer(entry.TagWeights ?? throw new InvalidOperationException(...))`
     - `Frequency` -> `new FrequencyScorer()`
     - `Reflexive` -> `new ReflexiveScorer()`
     - Default -> `throw new ArgumentOutOfRangeException()`

3. Add tests to `PipelineBuilderTests`:
   - `WithPolicy_NullPolicy_Throws` — ArgumentNullException
   - `WithPolicy_ValidPolicy_BuildsSuccessfully` — basic policy with 2 scorers, build succeeds
   - `WithPolicy_SetsSlicerFromPolicy` — verify Knapsack slicer is used when policy specifies it
   - `WithPolicy_SetsPlacerFromPolicy` — verify UShaped placer when policy specifies it
   - `WithPolicy_OverrideAfterPolicy` — `WithPolicy(p).WithPlacer(custom)` uses custom placer (last-write-wins)
   - `WithPolicy_WithQuotas_AppliesQuotas` — policy with QuotaEntry list, verify quota behavior
   - `WithPolicy_StillRequiresBudget` — WithPolicy without WithBudget, Build() throws "Budget is required"

4. Update `PublicAPI.Unshipped.txt` with `WithPolicy(CupelPolicy)` method.

**IMPORTANT:** The TagScorer null check in CreateScorer is defensive — ScorerEntry constructor already validates this. But belt-and-suspenders for the factory method is acceptable.

**IMPORTANT:** Do NOT add a new scoring path — WithPolicy uses the existing AddScorer() path. If the user previously called WithScorer(), Build() will correctly throw the "Cannot mix" error.
