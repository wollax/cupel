   - `PlacerType` in root namespace: `Chronological`, `UShaped`

2. Write `ScorerEntryTests` first (TDD), then implement `ScorerEntry`:
   - Sealed class in `Wollax.Cupel` namespace
   - Constructor: `ScorerEntry(ScorerType type, double weight, IReadOnlyDictionary<ContextKind, double>? kindWeights = null, IReadOnlyDictionary<string, double>? tagWeights = null)`
   - Properties: `Type`, `Weight`, `KindWeights`, `TagWeights` — all read-only with `[JsonPropertyName]`
   - `[JsonConstructor]` on the constructor
   - Validation:
     - `weight` must be finite and > 0 (throw `ArgumentOutOfRangeException`)
     - When `type == ScorerType.Tag`, `tagWeights` must not be null (throw `ArgumentException`)
     - `kindWeights` is optional for all types (KindScorer has defaults)
     - Store defensive copies of dictionaries (new Dictionary wrapping input)
   - Tests to write:
     - Valid construction with each ScorerType
     - Weight validation: zero, negative, NaN, Infinity all throw
     - Tag type without TagWeights throws
     - Tag type with TagWeights succeeds
     - Kind type with and without KindWeights both succeed
     - Defensive copy: mutating original dictionary doesn't affect entry

3. Write `QuotaEntryTests` first (TDD), then implement `QuotaEntry`:
   - Sealed class in `Wollax.Cupel.Slicing` namespace
   - Constructor: `QuotaEntry(ContextKind kind, double? minPercent = null, double? maxPercent = null)`
   - Properties: `Kind`, `MinPercent`, `MaxPercent` — all read-only with `[JsonPropertyName]`
   - `[JsonConstructor]` on the constructor
   - Validation:
     - At least one of minPercent/maxPercent must be specified (throw `ArgumentException`)
     - minPercent in 0-100 range if specified (throw `ArgumentOutOfRangeException`)
     - maxPercent in 0-100 range if specified (throw `ArgumentOutOfRangeException`)
     - If both specified, minPercent <= maxPercent (throw `ArgumentException`)
   - Tests to write:
     - Valid construction with min only, max only, both
     - Neither specified throws
     - Out-of-range values throw
     - Min > max throws

**IMPORTANT:** Follow existing codebase conventions:

- Test class naming: `{Type}Tests`
- Test namespace: `Wollax.Cupel.Tests.Policy` (new directory)
- XML doc comments on all public types and members
